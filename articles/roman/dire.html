简介

这个数独是交大 Speit 学院大一第一学期的项目。主要用的是 python 中的 pygame，架构尚待成熟。

实现的功能包括：

  1. 生成三种难度的数独谜题（80数，45数，25数）
  2. 用方向键或鼠标选中格子，用数字键填入
  3. 错误动态提示
  4. 显示解题时间
  5. 检测当前填法是否正解


我们的最好希望是把它做成一个界面优美的、代码结构清晰的理想的数独练习游戏，主要在 Windows 平台运行。
